    Drone-Assisted App

    Namo-Drone-Didi (Govt. Scheme)

    Built a Drone-Assisted Fertilization Monitoring Dashboard App The app provide drone-assisted fertilization by simplifying workflows, providing cost analysis, and delivering actionable insights. Designed with scalability and user-friendliness in mind, the platform bridges the gap between cutting-edge technology and practical farming solutions

    Languages & Technologies

    JavaScript , PHP, GoogleMapsAPI, Tableau, HTML, CSS and integrated Drone APIs for real-time data processing. Used Leaflet for geospatial mapping and Tableau for advanced data visualization to create interactive dashboards tailored to agricultural needs

    More Detail

    During my internship at National Fertilizers Limited (NFL), Noida, I developed the "Drone Didi" project, an app enabling rural women to operate drones for fertilizer spraying and revenue generation.

    • Integrated a drone API for automated field area calculation.
    • Developed a cost estimation module for optimized fertilizer use.
    • Designed a user-friendly interface with multilingual support.
    • Implemented real-time field mapping for precise fertilizer application.

    This project empowered rural women, enhanced agricultural efficiency, and aligned with smart farming initiatives. It combined my expertise in Geoinformatics and full-stack development for sustainable solutions.
